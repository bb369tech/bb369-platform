# BB369 Platform

A structured decision platform for service-based businesses.

This platform helps turn complex customer requests into clear options, allowing users to understand, compare, and choose services more easily.

---

## What This Platform Does

- collect customer requests  
- structure problems into A/B/C options  
- provide reference price ranges  
- guide users to book services  

---

## Core Flow

Upload → Select Problem → Compare Options → Book Service  

---

## Use Cases

This platform is designed for service businesses such as:

- windows and doors  
- cabinets and storage  
- home upgrade services  

---

## Platform Role

This repository represents the **application layer** of the BB369 system.

It focuses on real-world implementation and customer-facing service flows.
